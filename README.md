# KufarPro bot\n
KufarPro is a platform to get up-to-date information about advertisements on Kufar.by based on choosen filters. KufarPro supports such categories as: automobiles, real estate (flats, houses, garages) and other things (bicycles, mobile phones, TVs).\n
You can find these bots in Telegram.\n
Auto KufarPro - @auto_kufarpro_bot\n
Real estate KufarPro - @re_kufarpro_bot\n
Other things KufarPro - @other_kufarpro_bot\n
